---
title: container image list
description: List local images
---

Lists local images. Verbose output provides additional details such as image ID, creation time and full size; JSON output provides the same data in machine-readable form.

## Usage

```bash
container image list [--format <format>] [--quiet] [--verbose] [--debug]
```

## Options

<ParamField path="--format" type="string" default="table">
  Format of the output (values: json, table)
</ParamField>

<ParamField path="-q, --quiet" type="boolean">
  Only output the image name
</ParamField>

<ParamField path="-v, --verbose" type="boolean">
  Verbose output
</ParamField>
