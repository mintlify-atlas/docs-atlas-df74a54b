Shows detailed information about one or more networks.

<Note>Network commands are available on macOS 26 and later.</Note>

## Usage

```bash
container network inspect <networks> ... [--debug]
```

## Arguments

<ParamField path="networks" type="string[]" required>
  Networks to inspect
</ParamField>
